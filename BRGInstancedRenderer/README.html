BRG Instanced Renderer

Documentation

This repository hosts the documentation for BRG Instanced Renderer.

Built With

This documentation site is built using docfx. The documentation articles are located in articles/ and written in markdown format.

Getting Started

Installing docfx:

dotnet tool install -g docfx

Serving the docs on localhost:

docfx --serve

Adding New Articles

Create a new .md file in articles/, and link it in articles/toc.md.
